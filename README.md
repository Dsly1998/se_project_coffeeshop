# Practicum Coffee Shop

This is the second project of the Web Development program at Practicum. It was created using HTML and CSS, based on the design brief. It is a fun creative landing page for the Practicum Coffee shop.


## Project features

- Semantic HTML5
- Flexbox
- Positioning
- Flat BEM
- A custom form for reservations
- CSS animation and transform
